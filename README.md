# **Arduino-Based-Redundant-Transmission-for-Buoy-Networks**

Introduction
This is an innovative project that utilizes Arduinos to monitor and analyze critical oceanographic parameters, specifically ocean currents and temperature. Designed with reliability and continuity at its core, this system ensures uninterrupted data collection and transmission, even in challenging marine environments.

Project Description
This project represents the apex of an endeavor undertaken by a team selected by our college to compete in a prestigious oceanic competition against other eminent institutions. This repository documents my contributions to the project, focusing primarily on the development of the circuitry required to power all systems, including a failsafe mechanism, as well as the methodologies employed for transmitting data to an onshore station. It introduces an innovative dual-Arduino circuit design that allows for a seamless transition of control from one unit to another in cases of damage or malfunction. This failsafe mechanism significantly bolsters the reliability of the data collection process from oceanic buoys.

Key Features:
Dual-Arduino System: Two Arduino units work in tandem, with one serving as the primary unit and the other as a backup. The system automatically switches to the backup Arduino if it detects a failure in the primary unit.
Ocean Current Analysis: Utilizes sensors to measure and record ocean current patterns, providing valuable data for environmental monitoring and marine research.
Temperature Monitoring: Continuously measures water temperature, an essential parameter for understanding marine ecosystems and climate patterns.
Data Transmission: Collected data is transmitted back to a land-based station, enabling real-time monitoring and analysis.
Robust Design: Engineered to withstand the harsh marine environment, ensuring long-term operation and reliability.

Technology Stack
Arduino Microcontrollers: The core of the data collection and processing system.
Sensors: Advanced sensors for accurate measurement of ocean currents and temperature.
Communication Module: Enables wireless data transmission to the onshore station.
Application
This system is an invaluable tool for marine biologists, environmental researchers, and climate scientists. It offers a reliable method for collecting oceanographic data, which is crucial for understanding marine ecosystems, tracking climate change, and informing environmental policy.
